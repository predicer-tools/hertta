# Hertta

Hertta is a GraphQL server that orchestrates [Predicer](https://github.com/predicer-tools/Predicer),
an agent-based optimization model.
Via Hertta's API, it is possible to build a model for Predicer
and optimize it using current electricity market and weather forecast data.

See [Hertta Documentation](https://predicer-tools.github.io/hertta/index.html)
for installation instructions, usage examples and API reference.

> [!WARNING]
> **This project is currently under development and may not be fully functional.**
